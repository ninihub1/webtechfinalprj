Weather App Sample Project

Welcome Page

This is the welcome page, where you can get started with using our app to stay updated on the latest weather conditions. From here, you can proceed to the home page by clicking the button provided.

Home Page

On the home page, you'll find a search bar at the top where you can enter the name of any city to get its current weather information. The home page also features a settings button that takes you to the settings page, where you can switch the theme between light and dark modes as per your preference. Additionally, there is an add button that allows you to add a city to your favorites list for quick access. You'll also find the favorites button here, which will take you to the favorites page. The home page displays the weather, including hourly and daily forecasts, ensuring you have all the information you need at a glance.

Home Page (Light Mode)

The home page in light mode provides the same functionalities with a different theme.

Settings Page

The settings page allows you to customize the app to your liking. Change the theme from light to dark mode or vice versa, and adjust other settings to enhance your user experience.

Settings Page (Light Mode)

The settings page in light mode offers the same customization options with a different visual theme.

Favorites Page

The favorites page is dedicated to all the cities you've added to your favorites list. This allows you to quickly view the weather for multiple locations without having to search for them individually each time.
